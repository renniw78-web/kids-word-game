# Kids Crossword Word Game

儿童填字游戏项目

## 功能要求

1. 作成填字游戏和单词库结构，单词库可以自己增加进去。
2. 填字游戏采用填空的形式，直接在单词的空位上填写。
3. 在游戏页面上提供下拉菜单，让玩家选择词库，并提供“全部词库”选项。
4. 一个单词重复出现五次，每次需要填写的空格位置不同。
5. 如果单词填空正确，单词呈现绿色；如果错误，呈现红色。
6. 如果填错，单词出现次数增加（原本5次 + 错误次数）。
7. 每个单词停留5秒，如果答对立即进入下一个。
8. 尽量增加单词读音（Text-to-Speech）。
9. 在页面底部显示学习进度动画：
   caterpillar → pupa → butterfly。
10. 使用 HTML + JavaScript 编写。
